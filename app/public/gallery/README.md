# Gallery 圖片資料夾使用說明

此資料夾用於存放「過往紀錄」頁面的活動照片。

## 檔案命名規則

照片檔案應遵循以下命名格式：
```
{年份}-{編號}.webp
```

例如：`2025-1.webp`、`2024-3.webp`

## 需要的圖片清單

### 2025 年照片（6 張）
- `2025-1.webp` - 課程討論
- `2025-2.webp` - 團隊合作
- `2025-3.webp` - 實作練習
- `2025-4.webp` - 講師授課
- `2025-5.webp` - 成果發表

## 圖片規格建議

- **格式**：JPG
- **建議尺寸**：1200x900 像素（4:3 比例）
- **檔案大小**：建議壓縮至 500KB 以內，以優化網頁載入速度
- **品質**：清晰、光線充足、構圖適當

## 使用位置

這些圖片會顯示在以下頁面：
1. **首頁** - Gallery 區塊預覽（顯示前 6 張）
2. **過往紀錄頁面** (`/gallery`) - 完整照片牆（顯示所有照片）

## 新增照片

如需新增其他年份的照片，請按照以下步驟：

1. 將照片放入此資料夾，遵循命名規則
2. 編輯以下檔案，新增照片資料：
   - `app/components/gallery.jsx`（首頁預覽）
   - `app/app/gallery/page.js`（完整頁面）

3. 在 `galleryImages` 陣列中新增對應的物件：
```javascript
{
  src: "/gallery/2026-1.webp",
  alt: "2026 寒訓活動照片 1",
  title: "2026 XXX",
  description: "照片描述",
  year: "2026",
}
```

## 注意事項

- 照片需獲得參與者同意後才能使用
- 避免使用包含個人敏感資訊的照片
- 確保照片版權歸屬清楚
- 定期檢查並更新過時的照片
